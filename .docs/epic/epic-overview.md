## エピック概要

### 問題背景

多忙な社会人のコンクリート診断士受験者が抱える3つの主要課題：

1. **学習時間確保の困難**: 仕事後の疲労と時間制約
2. **教材携帯の負担**: 分厚い専門書・問題集の持ち運び
3. **学習効率の低さ**: 苦手分野の客観的把握が困難

### 解決策

**「最も重い専門書を、最も軽いスマートフォンの中に。」**

スキマ時間を活用した効率的な学習環境を提供し、学習進捗の可視化により非効率な学習からの脱却を支援する。

### MVP機能範囲

1. **シンプルなユーザー管理**: デバイスIDベースの軽量な仕組み
2. **問題演習機能**: 4択問題の表示・解答・正誤判定・解説表示
3. **学習進捗管理**: 年度別進捗表示・学習履歴保存
4. **AI生成コンテンツ**: Azure OpenAI Serviceによる解説文生成

## 技術概要

### アーキテクチャ

- **フロントエンド**: React Native + Expo SDK 53 + TypeScript
- **バックエンド**: Azure Functions + Azure Cosmos DB
- **AI連携**: Azure OpenAI Service (GPT-4)
- **ストレージ**: Azure Blob Storage
- **監視**: Azure Application Insights

### 主要技術決定理由

1. **Expo採用**: 開発効率性とクロスプラットフォーム対応
2. **Azure統一**: サービス間連携の簡素化とコスト最適化
3. **TypeScript採用**: コード品質向上と開発体験改善
