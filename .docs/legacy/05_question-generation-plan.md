# 問題生成計画書（100問作成）

## 概要

本ドキュメントでは、コンクリート診断士試験対策アプリのための100問を生成する詳細計画を定義します。著作権許諾が得られなかったため、AI（Azure OpenAI Service GPT-4）を活用してオリジナル問題を作成します。

## 目標

- **総問題数**: 100問（4択問題）
- **年度**: 2024年度相当として作成
- **品質基準**: 実試験レベルの難易度と専門性
- **検証**: コンクリート診断士有資格者による監修

---

## 問題作成の方針

### 1. 出題範囲の体系化

コンクリート診断士試験の公式出題範囲に基づき、以下の8つのカテゴリーに分類します。

| カテゴリーID | カテゴリー名 | 問題数 | 割合 |
|------------|------------|--------|------|
| CAT-01 | コンクリート材料学 | 15問 | 15% |
| CAT-02 | コンクリート構造設計 | 12問 | 12% |
| CAT-03 | 施工技術 | 10問 | 10% |
| CAT-04 | 劣化メカニズム | 18問 | 18% |
| CAT-05 | 診断技術・非破壊検査 | 15問 | 15% |
| CAT-06 | 補修・補強工法 | 15問 | 15% |
| CAT-07 | 維持管理計画 | 10問 | 10% |
| CAT-08 | 関連法規・基準 | 5問 | 5% |
| **合計** | - | **100問** | **100%** |

### 2. 難易度バランス

| 難易度 | 問題数 | 割合 | 説明 |
|--------|--------|------|------|
| 基礎 | 30問 | 30% | 基本的な知識・定義の確認 |
| 標準 | 50問 | 50% | 実務的な知識の応用 |
| 応用 | 20問 | 20% | 複合的な思考が必要な高度な問題 |

### 3. 問題形式の統一

全問題は以下の形式で作成します：

```typescript
{
  id: string,              // "2024-q001" 形式
  year: 2024,
  number: number,          // 1-100
  category: string,        // カテゴリーID
  difficulty: string,      // "basic" | "standard" | "advanced"
  text: string,            // 問題文（200-400文字）
  choices: [
    {
      id: string,          // "2024-q001-a" 形式
      text: string,        // 選択肢（50-150文字）
      isCorrect: boolean
    }
  ],
  correctChoiceId: string,
  explanation: string      // 解説（300-600文字）
}
```

#### ステップ0-4: 検索機能のテスト

### フェーズ1: RAGパイプライン構築（Week 2）

#### ステップ1-1: RAG検索クエリの設計
各テーマに対して適切な検索クエリを設計：

```typescript
interface SearchQuery {
  theme: string;              // 問題テーマ
  searchText: string;         // キーワード検索用
  semanticQuery: string;      // セマンティック検索用
  filter?: string;            // メタデータフィルター
  top: number;                // 取得する関連チャンク数
}
```

例:
```json
{
  "theme": "中性化のメカニズムと進行過程",
  "searchText": "中性化 炭酸化 水酸化カルシウム CO2",
  "semanticQuery": "コンクリートの中性化はどのようなメカニズムで進行するか",
  "top": 5
}
```

#### ステップ1-2: RAG対応プロンプトテンプレート作成
```
あなたはコンクリート診断士試験の問題作成の専門家です。
以下の参考資料に基づいて、問題を作成してください。

【参考資料】
{retrieved_chunks}

【作成する問題】
- カテゴリー: {category}

- 問題形式: 4択問題

【重要】
- 参考資料の内容に忠実に基づくこと
- 参考資料に明記されていない情報は推測しない
- 具体的な数値やデータは参考資料から引用する

【作成内容】
1. 問題文（200-400文字）
   - 参考資料の内容を反映
2. 4つの選択肢（各50-150文字）

```
