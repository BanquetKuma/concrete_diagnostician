## 📚 学習ポイント：なぜ法的ドキュメントが必要か

### App Store 審査での必須項目

```
【Apple が求める最低限の条件】

1. プライバシーポリシー
   - すべてのアプリで必須
   - URL でアクセス可能であること
   - 収集するデータを明記

2. サポート URL
   - ユーザーが問い合わせできる場所
   - 審査員もここを確認する

3. 利用規約（推奨）
   - 課金がある場合は必須
   - 無料アプリでも用意すべき

【ないとどうなる？】
❌ 審査でリジェクト
❌ アプリの信頼性低下
❌ 法的トラブルのリスク
```

### 個人情報保護の法律

```
【世界の主要な法律】

GDPR（ヨーロッパ）2018年〜
- 世界で最も厳しい個人情報保護法
- EU市民のデータを扱う場合は適用
- 違反すると売上の4%または2000万ユーロの罰金

CCPA（カリフォルニア）2020年〜
- 米国初の包括的プライバシー法
- カリフォルニア住民のデータが対象
- 「データを売らないで」という権利

日本の個人情報保護法（2022年改正）
- 6ヶ月以上保有するデータが対象
- 本人の同意なく第三者提供は禁止
- 漏洩時の報告義務

【MVPでの対応】
- 個人を特定できるデータは収集しない設計
- 収集するものは最小限に
- 何を収集するか明確に説明
```

### App Privacy ラベル（iOS 14以降）

```
【App Store に表示される情報】

┌─────────────────────────────────┐
│ App のプライバシー              │
│                                 │
│ ┌──────────┐ ┌──────────┐     │
│ │  追跡に   │ │  ユーザに │     │
│ │ 使用される│ │ 関連付け  │     │
│ │  データ   │ │られたデータ│     │
│ └──────────┘ └──────────┘     │
│                                 │
│ 詳細を表示 >                    │
└─────────────────────────────────┘

【3つのカテゴリ】
1. 追跡に使用されるデータ
   → 他社の広告に使われる（なしが望ましい）

2. ユーザに関連付けられたデータ
   → アカウントに紐づくデータ

3. ユーザに関連付けられないデータ
   → 匿名データ（デバイスIDなど）
```

## 7.4.1 プライバシーポリシー

### 必須項目

App Store 審査で確認される項目:

- 収集するデータの種類
- データの利用目的
- 第三者への共有有無
- データの保護方法
- ユーザーの権利
- 連絡先

### プライバシーポリシー テンプレート

```markdown
# プライバシーポリシー

最終更新日: 2024年12月17日

## はじめに

本プライバシーポリシーは、「コンクリート診断士試験対策」アプリ（以下「本アプリ」）における個人情報の取り扱いについて説明します。

## 収集する情報

本アプリは以下の情報を収集します：

### 自動的に収集される情報

- **デバイス識別子**: ユーザーの学習進捗を保存するため、デバイス固有の識別子を生成します
- **学習履歴**: 解答した問題、正誤結果、解答日時
- **アプリ使用状況**: 起動回数、使用時間

### 収集しない情報

- 氏名、住所、電話番号などの個人を特定できる情報
- 位置情報
- 写真、連絡先などのデバイス内データ

## 情報の利用目的

収集した情報は以下の目的で利用します：

- 学習進捗の記録と表示
- アプリの機能改善
- 不具合の修正

## 第三者への共有

本アプリは、法令に基づく場合を除き、収集した情報を第三者に提供しません。

## データの保管

- 学習履歴データはクラウドサーバー（Cloudflare, Turso）に保存されます
- データは暗号化された通信（HTTPS）で送受信されます
- サーバーは適切なセキュリティ対策を講じています

## データの削除

アプリを削除することで、デバイス上のローカルデータは削除されます。サーバー上のデータ削除をご希望の場合は、下記連絡先までご連絡ください。

## お子様のプライバシー

本アプリは、13歳未満のお子様から意図的に個人情報を収集しません。

## プライバシーポリシーの変更

本ポリシーは予告なく変更されることがあります。重要な変更がある場合は、アプリ内で通知します。

## お問い合わせ

プライバシーに関するお問い合わせは、以下までご連絡ください：

- メール: support@example.com
- サポートページ: https://example.com/support
```

## 7.4.2 利用規約

### 利用規約 テンプレート

```markdown
# 利用規約

最終更新日: 2024年12月17日

## 第1条（適用）

本規約は、「コンクリート診断士試験対策」アプリ（以下「本アプリ」）の利用条件を定めるものです。ユーザーは本規約に同意した上で本アプリを利用するものとします。

## 第2条（利用資格）

本アプリは、スマートフォンまたはタブレット端末をお持ちの方であれば、どなたでも利用できます。

## 第3条（禁止事項）

ユーザーは、以下の行為を行ってはなりません：

1. 本アプリの内容を無断で複製、転載、配布する行為
2. 本アプリの改変、リバースエンジニアリング
3. 不正アクセスやサーバーへの攻撃
4. 法令または公序良俗に反する行為
5. その他、運営者が不適切と判断する行為

## 第4条（知的財産権）

本アプリに含まれるコンテンツ（問題文、解説、デザイン等）の著作権は、運営者または正当な権利者に帰属します。

## 第5条（免責事項）

1. 本アプリは「現状有姿」で提供されます。運営者は、本アプリの正確性、完全性、有用性について保証しません。
2. 本アプリの利用により発生した損害について、運営者は責任を負いません。
3. 本アプリは試験対策の補助を目的としており、試験の合格を保証するものではありません。
4. 問題・解説の内容は細心の注意を払っていますが、誤りがある可能性があります。

## 第6条（サービスの変更・終了）

運営者は、事前の予告なく本アプリの内容を変更、または提供を終了することがあります。

## 第7条（規約の変更）

運営者は、必要に応じて本規約を変更することがあります。変更後の規約は、本アプリ内または公式サイトに掲載した時点で効力を生じます。

## 第8条（準拠法・管轄）

本規約の解釈は日本法に準拠し、紛争が生じた場合は東京地方裁判所を第一審の専属的合意管轄裁判所とします。

## お問い合わせ

本規約に関するお問い合わせは、以下までご連絡ください：

- メール: support@example.com
```

## 7.4.3 サポートページ

### サポートページ コンテンツ

```markdown
# サポート

## よくある質問（FAQ）

### Q: アプリが起動しません
A: 以下をお試しください：
1. アプリを完全に終了し、再起動
2. デバイスを再起動
3. アプリを削除して再インストール

### Q: 学習進捗がリセットされました
A: デバイスを変更した場合、学習履歴は引き継がれません。同じデバイスでの再インストール後は、データが復元されます。

### Q: 問題の内容に誤りがあります
A: 下記お問い合わせ先までご連絡ください。確認の上、修正いたします。

### Q: アプリを解約したい
A: 本アプリは無料です。アプリを削除するだけで利用を停止できます。

## お問い合わせ

### メール
support@example.com

### 対応時間
平日 10:00 - 18:00（土日祝日を除く）

### ご連絡の際のお願い
以下の情報を添えてご連絡いただけると、スムーズに対応できます：
- ご利用のデバイス（例: iPhone 15）
- iOS バージョン
- アプリのバージョン
- 問題の詳細な説明

## バージョン履歴

### v1.0.0（2024年12月）
- 初回リリース
- 100問の問題を収録
```

npx wrangler pages deploy ./docs --project-name=concrete-diagnostician-docs

3. Source で「Deploy from a branch」を選択
4. Branch で「main」、フォルダで「/docs」を選択
5. 「Save」をクリック
6. 数分後にURLが有効化される
